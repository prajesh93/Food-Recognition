1. Extract all the files from the zip folder into a single folder.

2. Run the file untitled1.m

3. A window will pop up.

4. Select the food joint you want to.

5. Select a number from the list of numbers in the drop down menu.Each number corresponds to one instance of the testing image. 

6. Then click on ‘Calorie Checker’. It will execute the process of creating the Database, training the SVM and also predicting the result.

7. The final result stating the food item and the calorie intake will be displayed in the text box.Along with the image of the tested data on the 
   side

8. Repeat steps 2 to 7 if you want to test another image, else click exit.
